# ODOP Market Linkage Platform

A full-stack web application to promote and sell regional district-based products.

## Installation

### Backend
1. `cd server`
2. `npm install`
3. Create `.env` from `.env.example`
4. `npm start`

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev`

## Roles
- **Buyer**: Browse and buy products.
- **Seller**: List and manage regional products (needs Admin verification).
- **Admin**: Verify sellers and view platform analytics.
